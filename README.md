### Preface
这个主题是在[yelee](https://github.com/MOxFIVE/hexo-theme-yelee.git)的基础上做了一些样式调整，使效果更加简洁。

### Installation 安装主题

```
git clone https://github.com/jinzhan/hexo-theme-breeze.git themes/breeze
```

修改Hexo根目录下的 _config.yml配置文件：

```
theme: breeze
```

### Update 更新

```
cd themes/breeze
git pull
```
